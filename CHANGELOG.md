# 0.5: Initial release

## Features

- round-trippable conversion
- macros expansion and substitution by adding to `@macros` dict
- importable python library
- incremental/streaming decoding
- jdot CLI tool
   - pretty-printing with `-p` / `--pretty`
- JDOT grammar in BNF
- add Apache License 2.0
